# StaticBitVec
簡潔ビットベクトルです。

## 定義
以下の操作を高速にします。
ここで与えられるビット列を 
$(A_i)_{i = 0}^{N - 1}$
とします。
$A$
は 
`bool`
の列とします。

### 長さ ( `len()` )
$N$
を返します。
計算時間は、
$O(1)$
です。

### アクセス ( `get_at(i)` )
$A_i$
を返します。
計算時間は、
$O(1)$
です。

### ランク ( `rank(e)` ) 
$$
\sum_{i = 0}^{e - 1} A_i
$$
を返します。
計算時間は、 
$O(1)$
です。
